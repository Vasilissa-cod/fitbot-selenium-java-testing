# Документация проекта FitBot Selenium Java Testing

Эта папка содержит документацию для проекта автоматизированного тестирования FitBot на основе FatSecret.

## Содержание

- **manual_test_cases.md** - Ручные тест-кейсы (TC001-TC008), которые были автоматизированы в проекте

## Структура тестов

Все ручные тесты были автоматизированы и разделены по классам:

- **FatSecretAuthTests.java** - TC001 (Регистрация), TC002 (Авторизация)
- **FatSecretDiaryTests.java** - TC003 (Добавление продукта), TC004 (Поиск продуктов)
- **FatSecretWeightTests.java** - TC005 (Отслеживание веса)
- **FatSecretUXTests.java** - TC006 (Адаптивность мобильной версии)
- **FatSecretPerformanceTests.java** - TC007 (Производительность загрузки)
- **FatSecretSecurityTests.java** - TC008 (Безопасность соединения)

## Связь между ручными и автоматизированными тестами

Каждый автоматизированный тест соответствует одному ручному тест-кейсу и реализует те же шаги и проверки, что описаны в ручных тестах.
